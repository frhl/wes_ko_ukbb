# Exome-wide evidence of compound heterozygous effects across common phenotypes in the UK Biobank
[![DOI](https://zenodo.org/badge/374424178.svg)](https://zenodo.org/doi/10.5281/zenodo.10926001)

This repository contains the computational pipeline designed to analyze compound heterozygous predicted loss-of-function genetic variations using Whole Exome Sequencing data from the UK Biobank. Importantly, the code contained in this repository can be used to reproduce and replicate the analyses presented in the associated preprint.
